# SAE501 - Synapse - Vincent ROURE, Mattéo BALDINETTI & Ken DIHN
## Contexte
Situation d'apprentissage et d'évaluation du 5e semestre, projet de réalisation d'un site complet pour une entreprise de formation en informatique.

## Technologies utilisées
- React 19.1.16
- Vite 7.1.12
- Bootstrap 5.3.8
- React Router DOM 7.9.5
- ChartJS
- CountUP

## Installation
	# Cloner le projet et y accéder
	git clone https://github.com/MatteoBaldinetti/SAE-501_Synapse
	cd SAE-501_Synapse
	
	# Télécharger les packages
	npm i
	
## Arborescence
	SAE-501_Synapse/
	│
	├── nodes_modules/			# Installed packages
	├── public/
	│	├── assets/        		# Reusable components
	│   │	└── images/        	# Contains every used images
	│   └── data/				# JSON Files
	│
	└── src/					# React/TypeScript source    
	    ├── components/			# TypeScript components
		├── types/				# Typed files
		└── pages/				# Web pages
	    
## Utilisation
Pour démarrer le serveur local, utilisez le script suivant :

    npm run dev
